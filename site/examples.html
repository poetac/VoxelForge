Examples gallery

Reproducible reference designs.

Each example ships with a SHA-256 receipt. Re-running on the same voxelforge revision produces a bit-identical STL. Drift in any output is a bug; the cascade catches it before it lands.

Status: one example currently lives in examples/ on main — the 5 kN reference. The five entries below it are roadmap placeholders pending Sprints 23–25 and the BB–3 corpus expansion. Tags reflect intended status, not shipped state.

Reproducibility receipts

Every example, byte-identical on rerun.

Each example folder contains receipt.json with: input parameters · voxelforge git SHA · PicoGK version · STL SHA-256 · gate fingerprint hash. The verify-examples CI job reruns every example on every PR and fails if any receipt drifts. This is the project's contract with itself.
