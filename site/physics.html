Heat transfer

Bartz, Dittus-Boelter, Gnielinski, Sieder-Tate.

Heat · gas-side Bartz convective coefficient
Bartz, JPL Memo 30-4 (1957); Sutton §5.5
h_g = (0.026 / Dt^0.2) · (μ^0.2 c_p / Pr^0.6)_0 · (P_c · g / c*)^0.8 · (Dt/r_c)^0.1 · (At/A)^0.9 · σ

Throat-anchored gas-side film coefficient. The σ factor accounts for property variation across the boundary layer; voxelforge uses Bartz's original form with film-temperature evaluation.

Validity
Conventional bell nozzles; throat-region heating
Uncertainty
±25–50 %
Gates
heat-flux-cap · regen-thermal-margin
Implementation
Core/HeatTransfer/Bartz.cs
Heat · coolant-side Dittus-Boelter
Dittus & Boelter (1930); Incropera §8.5
Nu = 0.023 · Re^0.8 · Pr^n n = 0.4 (heating) | 0.3 (cooling)

Smooth-tube fully-developed turbulent flow. voxelforge uses n=0.4 throughout regen channels (coolant is heated). Below Re=10⁴ falls back to Gnielinski; outside Pr ∈ [0.6, 160] falls back to Sieder-Tate.

Validity
Re > 10⁴ · 0.6 ≤ Pr ≤ 160 · L/D > 60
Uncertainty
±15–25 %
Gates
regen-thermal-margin · coolant-velocity-cap
Implementation
Core/HeatTransfer/CoolantSideHTC.cs
Heat · transitional Gnielinski
Gnielinski (1976); Incropera Eq. 8.62
Nu = (f/8)(Re−1000)Pr / [1 + 12.7(f/8)^0.5 (Pr^(2/3)−1)]

Used when Re drops into 3·10³–10⁴. friction factor f from Petukhov's smooth-tube fit. Tighter than Dittus-Boelter in the transitional band.

Validity
3·10³ < Re < 5·10⁶ · 0.5 < Pr < 2000
Uncertainty
±10 %
Gates
regen-thermal-margin
Implementation
Core/HeatTransfer/CoolantSideHTC.cs
Heat · wall material Bimetallic composite wall (series k, Voigt E)
Hibbeler §3.4 (series thermal resistance); A1 + Z1.2 hot-fix
k_eff = 1 / (t_liner/k_liner + t_jacket/k_jacket) E_eff = f·E_liner + (1−f)·E_jacket σ_y = min(σ_y,liner, σ_y,jacket)

Heat flows perpendicular to the wall, so thermal resistances add in series — the low-conductivity IN625 jacket (≈ 10–19 W/m·K) dominates, giving k_eff ≈ 13 W/m·K cold for a 25 % GRCop / 75 % IN625 wall (vs. 264 W/m·K under the prior parallel-blend error — a 20× overclaim). Hoop strain is tangential so stiffnesses add in Voigt (parallel) blend. Yield is governed by the weaker layer regardless of thickness ratio.

Validity
Wall index 4 (bimetallic GRCop-42 / IN625); single-material walls use direct tabulated k
Uncertainty
±5 % k at LPBF bond quality
Gates
YIELD_EXCEEDED · BURST_MARGIN_INSUFFICIENT · BIMETALLIC_BOND_ZONE_SHEAR
Implementation
Core/HeatTransfer/WallMaterial.cs

Combustion stability

Crocco N-τ and the chamber acoustic family.

Combustion Crocco N-τ instability criterion
Crocco & Cheng (1956); Harrje & Reardon (1972) NASA SP-194
N · |1 − e^(−iωτ)| < 1 → stable ωτ ∉ [π/2, 3π/2] for chug ωτ ∉ acoustic-mode windows for buzz/screech

Chamber treated as a 1D acoustic cavity with sensitive time-lag combustion. voxelforge solves the longitudinal, first-tangential, and first-radial modes — three gates, not one — and rejects designs where any mode falls in an unstable window.

Validity
L* > 0.6 m · injector pressure-drop ratio > 0.1
Uncertainty
Class-only
Gates
stability-chug · stability-1T · stability-1L
Implementation
Core/Combustion/Stability/CroccoNTau.cs

Nozzle & chamber geometry

Rao bell, isentropic flow, characteristic length.

Geometry Rao parabolic-bell contour
Rao (1958); Sutton §3.4
θ_n, θ_e = f(εₙ, L/L_15°) contour = parabola tangent to circular throat arc

80 %-bell default. Inflection angle θ_n and exit angle θ_e tabulated from Rao's curves; voxelforge fits a smooth parabola tangent to the throat circular arc and the exit angle line.

Validity
ε ∈ [3, 200] · 0.6 ≤ L/L_15° ≤ 1.0
Uncertainty
±2 % thrust loss vs ideal MOC
Gates
nozzle-overexpansion-margin · contour-curvature
Implementation
Core/Nozzle/RaoContour.cs
Geometry Characteristic length L*
Sutton §8.1; Huzel & Huang Table 4-1
L* = V_chamber / A_throat LOX/CH4: 1.0–1.3 m LOX/H2: 0.6–0.9 m LOX/RP1: 1.0–1.5 m

Combustion residence-time proxy. voxelforge keeps L* inside propellant-pair-specific bounds; below the lower bound, c* drops; above the upper, mass and heat load grow without ISP gain.

Validity
Tabulated propellants only
Uncertainty
Empirical band ±20 %
Gates
L-star-window · cstar-efficiency
Implementation
Core/Combustion/LStarLimits.cs

Feed system & turbopump

Cavitation, suction specific speed, shaft critical.

Fluid Suction specific speed S
Stepanoff (1957); Brennen, "Hydrodynamics of Pumps" §7
S = N · √Q / NPSH^(3/4) S_max ≈ 30,000 (rocket inducers)

Cavitation onset margin. Above S_max the inducer cavitates; voxelforge enforces S ≤ 28,000 by default with a 7 % safety factor. Lowering it forces a larger inducer or higher tank pressure.

Validity
Axial inducer + centrifugal stage
Uncertainty
±10 % cavitation onset
Gates
cavitation-margin · inducer-tip-speed
Implementation
Core/FeedSystem/Turbopump/Cavitation.cs
Structural Rotordynamic critical speed
Vance, "Rotordynamics of Turbomachinery" (1988) §3
N_op < 0.7 · N_crit_1 (subcritical) ω_n = √(k_eff / m_eff) / 2π

Lumped-mass shaft model. Bearings as linear springs, impellers as discrete masses. voxelforge keeps operating speed below 70 % of first critical (subcritical design); supercritical designs require a separate analysis path not yet implemented.

Validity
Stiff-bearing rigid-rotor approx
Uncertainty
±15 % critical-speed
Gates
shaft-critical-margin
Implementation
Core/FeedSystem/Turbopump/ShaftDynamics.cs
Structural · hoop stress Composite-cylinder hoop with isentropic gas P
Hibbeler §8.3 (composite cylinder); isentropic flow relations; Sprint G'
σ_hoop = ΔP · r / t_eff t_eff = t_liner + t_jacket P_gas = P_c · (1 + (γ−1)/2 · M²)^(−γ/(γ−1))

Gas-side liner and outer jacket share the circumferential hoop load in parallel — effective wall thickness is the sum of both. Gas pressure at each axial station is the local isentropic static value, which drops dramatically through the nozzle (exit ≈ 0.1 MPa at ε = 16 vs. P_c at the chamber). The prior constant-P_c path over-predicted exit-station hoop stress by 10–20×, producing false YIELD_EXCEEDED rejections on otherwise-feasible designs.

Validity
Thin-wall regime (r/t > 10); requires gasGamma > 0 for isentropic path
Uncertainty
±8 % (thin-wall approximation)
Gates
YIELD_EXCEEDED · BURST_MARGIN_INSUFFICIENT
Implementation
Core/Structure/StructuralCheck.cs

Citation discipline

Every correlation is implemented next to an XML doc-comment carrying the full citation (author, year, page or equation number). Validity-domain checks are runtime — the headless physics path will refuse to apply Dittus-Boelter at Re=300, not silently return a number that's wrong by 70 %.
