Day 4Apr 26Feasibility audit + 4 fixes

FA-018 — Feasibility-investigation audit

Audit + fix bundle · ADR-018

Five-phase investigation into the symptom "every canonical bench-sa preset returns 0 feasible candidates." Three production bugs found and shipped; two architectural gaps documented for follow-up sprints.

  • G_INJ ceiling units error: was 30–100× too low. Bench-sa now sees the full feasible band.
  • Face-thermal area metric wrong by 6×. Was using element pitch² instead of effective heat-transfer area.
  • AutoSeeder missing film-cooling defaults. Cold-start cases were rejected on WALL_TEMP before SA could explore.
  • AutoSeeder LRE-realistic params (was scaled for sub-kN class).

Result: RL-10 fixture now feasible on WALL_TEMP. Remaining 4 presets blocked on material-library gaps + missing film-T term in face-thermal model + NPSH model issues. ADR-018 documents the five prioritised follow-up sprints; each individually shippable.
